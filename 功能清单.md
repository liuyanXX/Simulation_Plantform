# 仿真平台功能清单

## 一、用户交互模块

### 1.1 模块概述

用户交互模块提供Web端用户界面和对应的FastAPI后端服务，支持用户通过浏览器与仿真平台进行交互，完成方案的录入、拆分、仿真、评估等全流程操作。

### 1.2 模块结构

```
user_interaction/
├── __init__.py
├── web_front/                  # Web端用户界面前端模块
│   └── index.html             # Vue.js单页面应用
└── web_action/                # 用户Web端服务子模块
    ├── __init__.py
    ├── main.py                # FastAPI主应用
    ├── models.py              # Pydantic数据模型
    └── routers/               # API路由模块
        ├── __init__.py
        ├── solution.py        # 方案管理路由
        ├── decomposition.py   # 方案拆分路由
        ├── simulation.py      # 仿真管理路由
        ├── evaluation.py      # 评估管理路由
        └── display.py         # 信息展示路由
```

## 五、设计原则

1. **面向对象设计原则**
   - 封装：每个模块通过API接口对外暴露功能
   - 单一职责：每个路由模块只负责一类功能
   - 依赖倒置：前端通过API与后端交互，不直接依赖后端实现

2. **微服务设计原则**
   - 模块独立：用户交互模块独立部署和运行
   - 松耦合：前后端通过RESTful API通信
   - 接口明确：每个API有明确的输入输出定义

3. **Python编码规范**
   - 遵循PEP 8编码规范
   - 使用类型注解
   - 模块和函数均有文档字符串

## 六、运行方式

### 6.1 启动后端服务

```bash
cd user_interaction/web_action
python main.py
```

服务启动后访问：
- 首页：http://localhost:8000/
- API文档：http://localhost:8000/docs

### 6.2 访问前端界面

直接用浏览器打开 `user_interaction/web_front/index.html` 文件，或使用本地服务器托管。
