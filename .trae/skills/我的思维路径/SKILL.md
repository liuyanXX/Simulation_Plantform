# 我的思维路径

## 功能描述

此技能用于记录您与Trae的交互内容，包括您的提问、Trae的回答以及中间的操作步骤，保存到"交互日志.log"文件中。

## 触发条件

当以下情况发生时调用此技能：
- 用户明确要求"记录交互"、"保存对话"、"记录思维路径"
- 用户想要查看之前的交互历史
- 用户要求"写日志"、"记录操作"
- 用户向Trae发送任意消息时，默认记录到日志文件

## 操作步骤

1. **准备工作**：确保项目根目录下存在"交互日志.log"文件，如果不存在则创建
2. **记录内容**：将用户的提问和Trae的回答追加写入日志文件
3. **格式规范**：
   - 记录时间戳（精确到秒）
   - 区分用户输入和系统输出
   - 包含操作类型和关键信息

## 日志格式

```
================================================================================
时间: YYYY-MM-DD HH:MM:SS
类型: 用户提问/系统回答/操作记录
内容: 
[详细内容]
--------------------------------------------------------------------------------
```

## 使用示例

### 示例1：记录用户提问
```
================================================================================
时间: 2026-06-22 20:30:00
类型: 用户提问
内容: 
帮我创建一个项目技能，技能名称叫"我的思维路径"
--------------------------------------------------------------------------------
```

### 示例2：记录系统回答
```
================================================================================
时间: 2026-06-22 20:30:15
类型: 系统回答
内容: 
已创建技能目录和SKILL.md文件，技能名称为"我的思维路径"
--------------------------------------------------------------------------------
```

### 示例3：记录操作记录
```
================================================================================
时间: 2026-06-22 20:30:05
类型: 操作记录
内容: 
创建目录: .trae/skills/我的思维路径
创建文件: SKILL.md
--------------------------------------------------------------------------------
```

## 注意事项

- 日志文件使用UTF-8编码，支持中文
- 日志文件位于项目根目录下：`交互日志.log`
- 每次交互追加写入，不覆盖历史记录
- 日志文件会自动创建，如果不存在的话
